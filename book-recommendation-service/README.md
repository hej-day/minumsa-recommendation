# 민음사 세계문학전집 추천 서비스 (Minumsa World Literature Recommendation Service)

방대한 민음사의 세계문학전집 중에서 무엇부터 읽어야 할지 모르는 고전문학 입문자들을 위해 개발된 미니멀하고 클래식한 웹 기반 추천 서비스입니다. 

사용자가 복잡한 탐색을 거치지 않고, 그날의 기분이나 원하는 깊이, 분량 등을 간단히 탭하는 것만으로 최적의 추천 도서 3권을 받아볼 수 있습니다.

---

## 1. 프로젝트 소개
- **목표**: 개인의 감성이나 난이도, 원하는 호흡의 길이에 맞는 민음사 세계문학전집 도서를 즉시 필터링하여 사용자에게 제안합니다.
- **디자인 컨셉**: 실제 민음사 세계문학전집 책 표지의 미니멀하고 단정 격자 무늬 디자인을 웹 요소를 통해 재해석하여, 클래식하고 차분한 모노톤에 버건디(#6B1824) 포인트를 적용한 고급스러운 분위기를 연출합니다.

---

## 2. 주요 기능
- **메인 페이지 (Home)**: 클래식한 독서 인용구와 서비스를 설명하는 웰컴 스크린을 제공합니다.
- **추천 조건 선택 (Quiz)**:
  - 읽고 싶은 분위기 (성장 이야기 / 사랑 이야기 / 철학적인 / 어두운 / 감성적인)
  - 선호 난이도 (입문용 / 보통 / 깊이 있는 독서)
  - 읽고 싶은 분량 (짧게 읽기 / 보통 / 긴 호흡)
- **예외 유효성 검사**: 문항 중 하나라도 선택하지 않고 추천 버튼을 누르는 경우 부드러운 경고 메커니즘을 적용하여 사용자 입력을 돕습니다.
- **추천 결과 카드 (Results)**: 선택 취향을 토대로 가중치를 반영한 알고리즘을 수행하여 일치율이 가장 높은 대표 도서 3권을 동적으로 추천 카드 UI로 노출합니다.
  - 실제 민음사 북커버와 유사한 그리드 레이아웃의 가상 북커버 제공
  - 난이도, 분량 태그 표시 및 작품별 명문장/한 줄 요약 제공
- **다시 추천받기 (Reset)**: 기존 입력된 취향 정보를 완전 초기화한 뒤 다시 선택 화면으로 되돌아갈 수 있는 기능을 지원합니다.
- **반응형 웹 지원 (Mobile Friendly)**: 데스크톱 화면뿐만 아니라 태블릿, 모바일 기기 등 모든 단말기 환경에서 화면이 흐트러지지 않는 반응형 레이아웃을 제공합니다.

## 3. 사용 기술
- **프론트엔드**: HTML5 (Semantic Tags), CSS3 (Vanilla CSS, Native Keyframes Animation), Modern JavaScript (ES6+)

## 5. 실행 방법
본 프로젝트는 외부 라이브러리 및 별도의 빌드 과정이 필요 없는 순수 정적 웹사이트입니다.

### 1) 로컬에서 바로 열기
1. 해당 리포지토리의 소스코드를 다운로드하거나 클론합니다.
2. `book-recommendation-service` 디렉터리로 이동합니다.
3. `index.html` 파일을 더블클릭하여 사용하는 웹 브라우저(Chrome, Safari, Edge 등)로 실행합니다.

### 2) 개발용 로컬 서버 활용하기
VS Code의 `Live Server` 확장 프로그램을 이용해 간편히 실행하거나, python 등을 이용해 아래와 같이 로컬 서버를 기동해 볼 수 있습니다.
```bash
# Python 3 환경인 경우
python -m http.server 8000
```
기동 후 브라우저에서 `http://localhost:8000` 주소로 접속합니다.
